# Web-Based Checkers

A modern, responsive, web-based Checkers game featuring a Player vs. Computer (PvC) mode. Test your skills against the built-in AI directly in your browser!

## Features

* **Player vs. Computer (PvC):** Play against a smart AI opponent.
* **Default Medium Difficulty:** The game begins at a balanced Medium difficulty for an immediate, fair challenge.
* **Smooth Animations:** Refined game pacing with delays before the computer's turn, indicators for the computer's moves, and fading animations for captured pieces.
* **Responsive Design:** Easily accessible and functional on both desktop and mobile devices.
* **Zero Dependencies:** Built entirely with pure HTML, Vanilla CSS, and JavaScript. No external libraries or build tools required.

## How to Play

Since this is a static web application, running the game is incredibly simple:

1. Clone the repository to your local machine:
   ```bash
   git clone <your-repository-url>
   ```
2. Navigate into the `Checkers` directory.
3. Open `index.html` in your favorite web browser.
4. Start playing against the computer!

## Project Structure

* `index.html` - The main structure and layout of the app.
* `css/styles.css` - Visual styling, layout for the board, and animation definitions.
* `js/main.js` - Application entry point and initialization logic.
* `js/game.js` - Core Checkers rules, board representation, and state management.
* `js/ui.js` - User interface handling, DOM manipulation, drag-and-drop/click events, and visual updates.
* `js/ai.js` - The decision-making logic for the computer opponent.

## Technologies Used

* HTML5
* CSS3 (Vanilla)
* JavaScript (ES6+ Vanilla)
